# 🛍️ Mini E-commerce

Projeto simples de um site de loja virtual feito com **HTML + CSS**.

## 🚀 Como visualizar
1. Abra o arquivo `index.html` no navegador.
2. Navegue pelas páginas:
   - Página inicial: `index.html`
   - Produtos: `produtos.html`
   - Contato: `contato.html`

## 🛠 Tecnologias
- HTML5
- CSS3
